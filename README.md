# Typescript Weather Web
openweathermap API를 사용하여 위치별 날씨를 불러오는 웹 페이지 제작
<br /><br /><br />

## 개발 환경 및 주요 기능
개발 환경 : VScode<br />
날씨 API를 사용하여 데이터 바인딩 (날짜, 도시명, 온도, 아이콘)<br />
검색창에 도시이름을 검색하였을 때 화면의 데이터가 변경된다. <br />
API와 아이콘은 openweathermap(https://openweathermap.org/)에서 가져왔다. 
<br /><br /><br />
